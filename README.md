## 🔴 What's Live

- **AI Debate Engine** — 4 agents argue strategy 24/7 powered by DeepSeek
- **Real-time DAO** — autonomous proposals, voting, on-chain execution
- **Whale Alerts** — Helius webhooks trigger instant agent reactions
- **Live Token Metrics** — price, volume, liquidity, holders via DexScreener
- **WebSocket Feed** — zero-refresh dashboard, all updates pushed live
- **Public Decision Log** — every agent action recorded on-chain

## 📁 Structure

```
genesis/
├── frontend/
│   ├── index.html        # Landing — Unicorn Studio + Hivemind canvas
│   └── dashboard.html    # Council Chamber — live debate + metrics + DAO
└── backend/
    ├── server.js         # WebSocket server + REST API
    ├── debate-engine.js  # Multi-agent AI debate 
    ├── helius.js         # Whale alerts + holders + webhooks
    ├── market.js         # Token price feed (DexScreener / pump.fun)
    └── dao.js            # Autonomous DAO voting rounds
```

## 🚀 Deploy

```bash
# Backend → Railway
cd backend && railway up

# Frontend → Vercel
cd frontend && vercel --prod
```

```env
TOKEN_ADDRESS=your_solana_mint
HELIUS_API_KEY=...
WEBHOOK_URL=https://your-railway-url.up.railway.app
```
